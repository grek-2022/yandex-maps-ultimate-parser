# yandex-maps-ultimate-parser
Мощный парсер Яндекс.Карт с автокликом по скрытым телефонам, сбором рейтинга, отзывов, сайтов и соцсетей. Поддерживает бесконечную прокрутку и обход капчи 

Как пользоваться
git clone https://github.com/grek-2022/yandex-maps-ultimate-parser.git
cd yandex-maps-ultimate-parser
pip install -r requirements.txt
playwright install chromium
python yandex_parser_ultimate.py


# 🗺️ Yandex Maps Ultimate Parser

[![Python 3.9+](https://img.shields.io/badge/python-3.9+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Мощный, надёжный и полностью автоматический парсер Яндекс.Карт** с обходом капчи, кликом по скрытым телефонам и сбором всех ключевых данных об организациях.

---

## ✨ Возможности

- ✅ Автоматический сбор организаций по поисковому запросу и городу
- ✅ Загрузка **всех результатов** (бесконечная прокрутка)
- ✅ Извлечение полей:
  - ID организации
  - Название
  - Адрес
  - Координаты (широта, долгота)
  - Категория
  - Рейтинг (числовой)
  - Количество отзывов
  - Телефон(ы) – включая скрытые за кнопкой «Показать телефон»
  - Сайты
  - Социальные сети (Telegram, WhatsApp, VK, YouTube и др.)
  - Прямая ссылка на карточку
- ✅ Обход капчи с ручным вмешательством (один раз за сессию)
- ✅ Эмуляция человеческого поведения (случайные задержки, прокрутка)
- ✅ Сохранение результатов в CSV (UTF-8 with BOM – для Excel)
- ✅ Не требует ручного указания драйверов (использует Playwright)

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/ваш_username/yandex-maps-ultimate-parser.git
cd yandex-maps-ultimate-parser

2. Установите зависимости
bash
pip install -r requirements.txt
playwright install chromium
3. Запустите парсер
bash
python yandex_parser_ultimate.py
4. Введите параметры
text
Поисковый запрос: кафе
Город: Москва
Максимум организаций (0 - все): 100
Парсер откроет браузер, автоматически прокрутит страницу, соберёт ссылки, а затем начнёт парсить каждую организацию.

 Пример выходных данных (CSV)
ID организации	Название	Адрес	Координаты	Категория	Рейтинг	Отзывов	Телефоны	Сайты	Соцсети	Ссылка

Как это работает
Playwright открывает браузер и переходит на страницу поиска.

Скрипт эмулирует прокрутку, чтобы подгрузить все доступные карточки.

Собираются уникальные ссылки на страницы организаций.

Для каждой ссылки:

Открывается карточка.

При необходимости нажимается кнопка «Показать телефон».

Из HTML и JavaScript-скриптов извлекаются все поля.

Делается случайная пауза (5–8 секунд) – для имитации человека и снижения риска блокировки.

Все данные сохраняются в CSV-файл.

🛡️ Обход капчи
Если Яндекс показывает капчу, парсер:

Останавливается.

Выводит сообщение в консоль.

Ждёт, пока вы решите капчу вручную в браузере.

После нажатия Enter продолжает работу с того же места.

Совет: При первом запуске может появиться капча – решите её один раз, и сессия продолжит работать без повторных остановок.

📦 Зависимости

text
playwright
beautifulsoup4
pandas
Установка одной командой: pip install -r requirements.txt
