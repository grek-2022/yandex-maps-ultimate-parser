# yandex-maps-ultimate-parser
Мощный парсер Яндекс.Карт с автокликом по скрытым телефонам, сбором рейтинга, отзывов, сайтов и соцсетей. Поддерживает бесконечную прокрутку и обход капчи 

Как пользоваться
git clone https://github.com/grek-2022/yandex-maps-ultimate-parser.git
cd yandex-maps-ultimate-parser
pip install -r requirements.txt
playwright install chromium
python yandex_parser_ultimate.py
